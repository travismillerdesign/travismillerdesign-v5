================================================================================
QUICK FIX: 3 STEPS
================================================================================

1. Create .editorconfig:
   root = true
   [*]
   indent_style = space
   indent_size = 2
   end_of_line = lf

2. Create .prettierrc:
   {
     "useTabs": false,
     "tabWidth": 2
   }

3. Run Prettier:
   npm install -D prettier
   npx prettier --write "src/**/*" ".eleventy.js" "optimize-images.js"
