Leia Inc

Unifying design language for 3D technology

Case Study —

    My role

  • Brand refresh (logo refinement, wordmarks, color system)
  • Complete component library optimized for 3D viewing
  • App icon family
  • Documentation and implementation guidelines

The challenge

With the Lume Pad 2 launch approaching, Leia needed a unified design system and cohesive brand identity. The unique challenge: no one had systematically designed for 3D displays before. Every component needed to work in 3D space with appropriate depth, contrast, and scale.

Brand Evolution

Refined logo proportions, created custom product wordmarks, and established a consolidated color palette with semantic variables.

Design Foundation

Working within an 8pt grid informed by the tablet's resolution limitations, I defined how components should behave in 3D space, appropriate depth, contrast, and scale for lightfield viewing.

Build Component Library

I used Material Design as the foundation (Android platform) and customized it for 3D requirements. Designing for 3D meant adapting Material Design—buttons needed more contrast than guidelines recommended because depth perception changes how users see color. This required balancing established patterns with novel constraints. Built navigation, buttons, media controls, typography, and icons—all documented with clear guidelines for designers and engineers.

App Icons

Redesigned the icon family so each clearly communicates its app's function while maintaining visual unity. For example, the LeiaChat icon represents "video with depth" using a camera's profile angle.

Implement and Enable

Rolled out the system app-by-app with phased implementation. Conducted 1:1 working sessions with the design team and collaborated with engineering to establish design-code parity. Presented to stakeholders weekly to maintain alignment.

Solution

    Component Library
  • Navigation: Sidebar approach optimized for landscape tablet use
  • Buttons: Unified variations with flexible white space
  • 3D Media Controls: Modular system with appropriate scale and contrast
  • Typography: Consistent size ratios with baseline grid
  • Icons: Extended Material library with consistent construction
Brand System

Refined logos with clear usage guidelines, custom product wordmarks with consistent stroke weights and proportions, and a consolidated color palette with semantic variables for Material Design layering.

Visual Language

Gradient system using Cinema4D prism concepts to represent the optical technology, plus device animations and default wallpapers for a cohesive branded experience.
